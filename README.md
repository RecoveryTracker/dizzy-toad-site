# Dizzy Toad Woodworks — website

Static gallery/portfolio site for woodturner **Victor Todd** ("Dizzy Toad"),
Hendersonville NC. Live at **https://dizzytoadwoodworks.netlify.app**.

No build step — plain HTML/CSS/JS. Netlify auto-deploys every push to `main`.

## Files
- `index.html` — the whole site (nav, hero, gallery, lightbox, shows, about, contact, QR code)
- `pieces.js` — the catalog (`window.PIECES`); regenerated by the in-page Admin → Publish
- `admin.js` — self-contained admin CMS (soft password gate; edit/add/sell/upload photos)
- `photos/` — product photos
- `_headers` — Netlify security headers
- `robots.txt` — crawler policy (indexing welcome)

## Update the catalog
1. On the live site, open the **🐸 Admin** panel and make changes, then **Publish** (downloads a fresh `pieces.js`)
2. Replace `pieces.js` here with that file
3. `git add -A && git commit -m "Update catalog" && git push` — Netlify deploys in seconds

The published `pieces.js` is always authoritative; any stale browser cache is auto-discarded.
